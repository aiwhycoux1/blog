Creating Whiteboard Animations With jsWritingTool

Written by Ashraff Hathibelagal • 30 November 2016

  Design

Introduction

Most of us are familiar with the handwritten animation effect, which can be seen in many Youtube videos—mostly videos that are explaining something. Creating such videos has always meant using tools that are quite expensive. Therefore, I decided to create a free tool that requires no registration and no downloads whatsoever. Of course, it isn’t as powerful as the commercial alternatives. However, I think it gets the job done, if you aren’t creating anything professional. It’s called jsWritingTool, and all it needs is a browser.

Here’s a YouTube video that shows you, in detail, how to use jsWritingTool. I strongly suggest that you watch it before using the tool.

Please go ahead and give jsWritingTool a spin. If you have any issues, leave a comment on this page.
